# Hotel Management System (Java CLI)

## Overview
A console-based hotel management system built in Java as a first-semester project.  
It simulates real-world hotel operations including booking, food ordering, services, billing, and receipt generation using file-based storage.

---

## Features
- User registration and login system
- Admin and user roles
- Room booking with availability tracking
- Food ordering system
- Room services module
- Billing system with GST calculation
- Receipt generation
- File-based persistent storage

---

## Tech Stack
- Java
- File Handling (Java I/O)
- CLI (Command Line Interface)

---

## What I Learned

- Handling file-based persistence
- Managing program state in large procedural code
- Designing menu-driven systems
- Basic system simulation (real-world logic mapping)
